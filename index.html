Random Words is a simple vocabulary builder. Words are randomly selected from a file that contains over 370,000 English words (file can be found here). Beautiful Soup was implemented to scrape definitions from Merriam-Webster’s website. If a selected word is not in Merriam-Webster’s dictionary, the word is replaced with the first suggested word. The program also allows users to quiz themselves on previously seen words.
